Projeto Meu site pessoal

Este projeto foi desenvolvido visando expandir meu conhecimento em desenvolvimento Web.

Aqui você vai encontrar algumas das tecnologias que estive estudando nos últimos dias, eu usei o planejamento estratégico criado como atividade para o professor Junior como base.

Objetivo: Transição de carreira

    Metas

  1. Meta 1: Estudar sobre desenvolvimento de software, incluindo cursos, projetos pessoais, bootcamps durante 2 meses.
  2. Meta 2: Estudar sobre deploy de sistemas, Cloud computing, Devops (Docker principalmente) durante 1 mês.
  3. Meta 3: Aprimorar skills de algoritmos avançados com prática e leitura de livros, durante 3 meses em conjunto com a meta 1 e 2.

Pontos fortes

Hard skills: programação back-end, bancos de dados SQL e NoSQL (MySQL, PostgreSQL, MongoDB), desenvolvimento de API's, inglês avançado.

Soft skills: Comunicação clara e assertiva, Capacidade de resolver problemas, análise de eficiência de implementações, facilidade em trabalhar com equipes diversas, manter a calma em situações de pressão, iniciativa, liderança.

Pontos fracos

Hard skills: Algoritmos avançados, integração de sistemas back-end + front-end, DevOps (Docker, Kubernetes), Cloud computing, conhecimento em IA's.

Soft Skills: Dificuldade em expor ponto de vista em grandes grupos, gerenciamento de tempo, procrastinação, criatividade.
